Image and Link in HTML

HTML Links

Hyperlinks Syntax : The HTML < a > tag defines a hyperlink. It has the following syntax

< a href ="url" > link text < /a>

Example: < href ="https://www.w3schools.com/" < /a> Visit w3schools

Images in HTML

  1. Set the background image for the page

    2. < body background = "file_name" >< /body>

  2. Insert images into Folder

    3. < img src = "link_to_folder" title = "Note" >

  3. Insert images into Web pages

    4. < img src = "Copy_link_address" tile = "Note" >
